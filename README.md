
# Uber Eats
A cross Platform Application for online food ordering 
Where User can book Food from his favorite Restornat
Using React JS
## Installation

Install my-project with terminal 

```bash
  git clone https://github.com/apartyagi/uber-eats.git
```

  cd uber-eats
